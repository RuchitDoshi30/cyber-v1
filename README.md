# Cyber Forensics Investigation Dashboard

A web-based frontend application designed to simulate a cyber forensics investigation system.

This project focuses on **visualizing, investigating, and reporting cyber logs** collected from multiple sources such as operating systems, applications, servers, and IoT devices.

The application is **frontend-focused** and demonstrates the investigation workflow rather than backend or machine learning implementation.
